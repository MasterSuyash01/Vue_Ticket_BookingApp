# Getting Started

To begin using the project, follow these steps:

## Installation

1. Navigate to the `backend` directory: `cd backend`
2. Install backend dependencies: `pip install -r requirements.txt`
3. Navigate to the `frontend` directory: `cd ../frontend`
4. Install frontend dependencies: `npm install`

## Usage

1. Start the Flask backend: In the `backend` directory, run `python main.py`.
2. Start the Vue.js frontend: In the `frontend` directory, run `npm run serve`.
3. Access the application: Open your web browser and go to `http://localhost:8080`.

Feel free to explore the application and start managing show bookings and ticket reservations!
